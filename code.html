What is TSL?

TSL is an object oriented numerical library written in C++. The library contains a selection of mathematical routines for solving a variety of problems. The linear algebra routines in TSL are powered by the Eigen C++ template library, Eigen provides fast and reliable solution of both dense and sparse linear systems and eigenvalue problems.

Downloading TSL

As TSL is still in development it is changing quite regularly. In order to get the most up to date version it is best to download or clone the GitHub repository. The repository can be downloaded from GitHub or if you already have git installed then the repository can be cloned using the terminal command 


          git clone https://github.com/tonywilliams1990/TSL

Using TSl

In order to use TSL you must have a recent C++ compiler such as GCC which is compatible with at least the C++98 standard. TSL also uses SCons and Python in order to build the code so these too must be installed. In order to compile the code simply cd to the TSL directory and type the command 


          scons
into the terminal. The initial compilation may take some time while the TSL library is compiled but this only needs to be done once. If you have make installed on your machine you can use the usual make commands with TSL such as 

          make
and 

          make clean
It is simple to add your own code to TSL, you just need to put your code somewhere in the Example directory and update the SConscript file as appropriate.

Motivation

TSL is meant to be an educational project in the sense of developing knowledge of C++ programming and the construction of numerical routines. TSL is not meant to be an optimised library for solving industrial strength problems. Since the underlying linear algebra in TSL is powered by the Eigen library reasonable performance can be expected for many problems however this is by no means guaranteed.

Thank you for your interest in TSL, please feel free to develop TSL further or make suggestions for its improvement.
