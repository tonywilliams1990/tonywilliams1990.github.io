Detailed Description

The Falkner-Skan equation

\[ f'''(\eta) + f(\eta)f''(\eta) + \beta \left[1 - \left(f'(\eta) \right)^2 \right] = 0 \]

is solved on \( \eta \in [0,\infty) \) subject to the boundary conditions \( f(0) = f'(0) = 0\) and \( f'(\eta) \to 1 \) as \( \eta \to \infty\).

The system is solved using rational Chebyshev functions on a semi-infinite interval \( TL_i(\eta; L) \). The rational chebyshev functions all have zero derivative as \( \eta \to \infty \) so it is necessary to use the change of variables \( u(\eta) = f(\eta) - \eta \). The Falkner-Skan system is transformed to

\[ u''' + (\eta + u)u'' - \beta u'\left(2 + u' \right) = 0 \]

subject to the boundary conditions \( u(0) = 0\), \( u'(0) = -1 \) and \( u' \to 0 \) as \( \eta \to \infty \). This nonlinear equation is solver using Newton iteration on the spectral coefficients.
