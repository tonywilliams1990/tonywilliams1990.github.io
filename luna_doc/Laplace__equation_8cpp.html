Detailed Description

Solve Laplace's equation

\[ \nabla^2 u = 0, \]

on the unit square \( (x,y) \in [0,1] \times [0,1] \).

The equation is subject to the boundary conditions

\[ u(x,0) = 0, \hspace{0.5cm} u(x,1) = \frac{1}{(1+x)^2 + 1}, \hspace{0.5cm} 0 \leq x \leq 1, \]

\[ u(0,y) = \frac{y}{1+y^2}, \hspace{0.5cm} u(1,y) = \frac{y}{4+y^2}, \hspace{0.5cm} 0 \leq y \leq 1. \]

The exact solution is given by

\[ u(x,y) = \frac{y}{(1+x)^2 + y^2}, \hspace{0.5cm} 0 \leq x,y \leq 1. \]

The equation is discretised using a second order finite-difference scheme with step size \( \Delta x = \Delta y = 1 / N \) where \( N \) is the number of intervals. The internal nodes in the discretisation are governed by the finite-difference equation

\[ u_{i-1,j} + u_{i,j-1} - 4u_{i,j} + u_{i,j+1} + u_{i+1,j} = 0, \]

where \( u(x,y) \approx u(x_i,y_j) = u_{i,j} \) and \( x_i = i \Delta x, \hspace{0.2cm} y_j = j \Delta y \) with \( i,j = 0,1,\ldots,N. \)
